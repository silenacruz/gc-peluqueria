**GC Peluquería** es una landing page minimalista y de alto impacto visual desarrollada para un estudio de estilismo profesional. El proyecto se centra en la conversión y en proyectar una identidad de marca moderna, sofisticada y *genderless* a través de una experiencia de usuario optimizada.

🔗 **Demo en vivo:** [Visitar Sitio Web](https://silenacruz.github.io/gc-peluqueria/)

---

## 🎯 ¿Qué es este proyecto?

Es una solución digital enfocada en el sector *beauty & wellness*. El objetivo principal es guiar al usuario a través de la propuesta de valor del estudio hasta el llamado a la acción (CTA) principal: la reserva de turnos, eliminando cualquier fricción en el proceso de venta.

## 🛠️ Reto Técnico

El mayor desafío fue lograr una experiencia de usuario fluida y una estética de "autor" manteniendo un código limpio. 

* **Arquitectura:** Implementación de componentes escalables y mantenibles.
* **Layout Moderno:** Uso intensivo de **CSS Grid y Flexbox** para un diseño responsivo real, priorizando el tráfico desde dispositivos móviles (*Mobile-First*).
* **Automatización:** Integración de la **WhatsApp Business API** para automatizar el flujo de reserva de turnos, permitiendo que el usuario agende directamente desde la interfaz sin pasos intermedios innecesarios.

## 🚀 Tecnologías Utilizadas

* **HTML5:** Estructura semántica avanzada para SEO.
* **CSS3:** Animaciones fluidas, variables (Custom Properties) y diseño adaptativo.
* **JavaScript (Vanilla):** Lógica de interacción y manejo de eventos de reserva.
* **GitHub Pages:** Deployment y hosting.
